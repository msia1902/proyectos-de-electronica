# Proyecto LED Simple

## Descripción

Este es mi primer proyecto práctico documentado en GitHub como parte de mi portafolio de electrónica.

El objetivo del proyecto es encender un LED utilizando una fuente de alimentación y una resistencia para limitar la corriente y evitar que el LED se dañe.

Este circuito es uno de los más básicos en electrónica y permite comprender conceptos fundamentales como corriente, voltaje y resistencia.

---

## Componentes utilizados

* 1 LED
* 1 Resistencia (220Ω o 330Ω)
* Fuente de alimentación (batería o fuente DC)
* Protoboard
* Cables de conexión

## Funcionamiento

El LED se conecta en serie con una resistencia.
La resistencia limita la corriente que circula por el LED para evitar que se queme.

Cuando se conecta la fuente de alimentación, la corriente fluye a través de la resistencia y el LED, provocando que el LED se encienda.

---

## Conceptos aprendidos

En este proyecto se aplican los siguientes conceptos básicos de electrónica:

* Ley de Ohm
* Limitación de corriente en LEDs
* Conexión de componentes en serie
* Uso básico de protoboard
