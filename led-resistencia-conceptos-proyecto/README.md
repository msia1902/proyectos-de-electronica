# Proyecto LED Básico

Circuito que enciende un LED usando una resistencia.

## Componentes
- LED
- Resistencia 220Ω
- Batería 9V
- Protoboard

## Funcionamiento
Cuando se cierra el circuito, el LED se enciende.

## Aprendizajes
- Uso de resistencias
- Control básico de LEDs
