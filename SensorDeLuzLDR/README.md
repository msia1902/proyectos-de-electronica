# Control de Iluminación Automático (LDR)

## Descripción
Este proyecto consiste en un sistema que detecta el nivel de luz ambiental y enciende un LED automáticamente cuando la luminosidad baja de un umbral específico. Es una simulación básica de cómo funcionan los postes de luz en la calle.

## Componentes utilizados
* Arduino Uno
* Sensor Fotoresistencia (LDR)
* Resistencia de 10k ohms (para el divisor de tensión)
* LED y resistencia de 220 ohms
